# To build sever and client: 
``````
Run make
``````

# To run server: 
``````
Run ./main -p [port]
Note: port >= 1024
Example: ./main -d -p 2023
``````

# To run client: 
``````
Run ./client -p [port] -i [sensorID] -a [IP]

example: ./client -p 2023 -i 4 -a 192.168.15.144
``````

# To use debug mode: 
``````
Add option -d 
``````
